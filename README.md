This project is a responsive website built for a real Brazilian Jiu-Jitsu gym, Carlson Gracie Irvine. The goal of this project was to create a clean, user-friendly platform that provides essential information for potential and current members, including class programs, schedules, instructors, and contact details.

This project demonstrates my ability to design and build a functional front-end application with real-world use cases.

Features:
  -Programs & Schedule Section – Displays available classes and training times
  -Instructor Profiles – Highlights coaching staff and credentials
  -Facility Overview – Showcases gym environment and amenities
  -Location & Contact Info – Easy access for new members
  -Responsive Design – Works across desktop and mobile devices
  -Contact Form Integration – Allows users to reach out directly
